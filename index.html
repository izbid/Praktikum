Day 2: OS & Networking

Infrastructure Focus: Getting the machine online.

1. Create Bootable USB INFRA
  1. Insert a clean USB drive (8GB+).
  2. Go to the Official Windows Download Page.
  3. Download the "Media Creation Tool".
  4. Run it and select "Create installation media" for your USB.
2. Install Windows 11 INFRA
  1. Plug the USB into your new build and boot it up.
  2. Enter BIOS (F2, F12, or DEL) to change the boot order to USB first.
  3. Follow the setup steps and select your primary storage drive.
Official Microsoft Guide
3. Network Connectivity Test NET
  1. Search for "cmd" in the Start menu and open it.
  2. Type ipconfig to find your local IP address.
  3. Type ping 8.8.8.8 to verify internet access.
